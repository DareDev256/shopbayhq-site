# ShopBayHQ — Marketing Site

Marketing landing page for [ShopBayHQ](https://shopbayhq.com), a shop-floor operating system for auto service shops.

## Stack

- **Next.js 16** (App Router, static export)
- **React 19**
- **Tailwind CSS 4**
- **Framer Motion** (scroll-triggered animations)
- **Inter** (Google Fonts via `next/font`)

## Sections

Hero → Stats strip → Problem → Features (4-card grid) → Founder proof → Pricing (3-tier) → FAQ (accordion) → CTA → Footer

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed to Vercel. Pushes to `main` auto-deploy.
